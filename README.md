# Right Hire - Car Rental & Lease Management System

A comprehensive Frappe application for managing car rental and lease operations.

## Installation

```bash
bench get-app right_hire
bench --site [site-name] install-app right_hire
bench --site [site-name] migrate
```

## Features

- Vehicle Fleet Management
- Reservations & Rental Agreements
- Lease Contracts
- Maintenance Tracking
- Financial Reports
- Utilization Analytics
